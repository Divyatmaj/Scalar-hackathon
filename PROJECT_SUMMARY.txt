🔄 RL LOOP IMPLEMENTATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STATE:      Interview question + metadata (difficulty, keywords)
            ↓
ACTION:     AI-generated answer (from LLM agent)
            ↓
REWARD:     Score-based evaluation (+10, +5, 0, -5)
            ↓
FEEDBACK:   Detailed analysis of missing concepts
            ↓
RETRY:      Improved answer if reward < 5 (automatic learning!)
